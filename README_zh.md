# 自激跳跃对期权定价是否重要？

**Hawkes–Kou 仿射定价、近似误差与经济显著性研究**

> **研究状态：进行中。** 本仓库是公开研究日志，不是生产定价器或交易策略。弱自激近似目前属于**形式渐近推导**，尚未完成余项界与完整收敛证明。

## 核心问题

Hawkes 自激相对于普通 Kou 跳跃，何时能够提供统计上可识别、数值上可靠且经济上显著的增量价值？

模型阶梯为：

1. Black / EWMA 基准；
2. Kou 跳跃扩散；
3. Hawkes–Kou；
4. 形式弱自激近似。

项目不预设复杂模型一定更好，而是比较定价精度、Greeks、校准稳定性与计算成本。

## 研究路径

项目最初使用指数记忆泛函与跳跃脉冲构造路径依赖波动率模型。滚动样本外测试发现，该模型在多数平静状态中弱于 EWMA。主要问题包括：连续扩散算子没有显式跳跃风险溢价、参数共线、阈值不稳定，以及无法稳定生成偏斜和微笑。因此研究转向显式刻画跳跃聚集与非对称厚尾的 Hawkes–Kou 框架。

## 严谨性声明

早期探索性公式以 Black 有效方差替代零阶 Kou 价格，并用高阶 Greeks 表示修正。该公式仅作为 **legacy 错误分析案例**保留。

已确认的两个关键问题：

1. 无自激时仍然是完整 Kou 跳跃模型，Black 矩代理会丢失偏度与尾部信息；
2. 原一阶时间核缺少一次时间积分，显著高估短期限自激修正。

当前形式展开为

\[
V^\delta=V_0+\delta V_1+O(\delta^2),
\]

但尚不能视为已经证明的定理。项目正在使用仿射特征函数、COS/Fourier 和带置信区间的事件驱动 Monte Carlo 重新建立基准。

详见 [`docs/known_issues.md`](docs/known_issues.md)。
